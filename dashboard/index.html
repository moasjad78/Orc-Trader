Deployment Steps
01 Upload the orb-webhook.zip to a new GitHub repo
02 Go to railway.app → New Project → Deploy from GitHub
03 Add environment variables: RH_EMAIL and RH_PASSWORD
04 Copy your Railway public URL and paste it above
05 Set up TradingView alerts using the templates on the right →

Strategy Settings
Account & Auth
Credentials are stored as RH_EMAIL and RH_PASSWORD environment variables in Railway — never in code or logs. The server re-authenticates automatically at 9:00 AM ET each day.
Add your Railway URL to UptimeRobot (free) to get SMS/email alerts if the server goes down.
